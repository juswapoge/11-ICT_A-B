Generation Of Computer

1st Generation

The Computers of first generation used vacuum as the basic components to memory and circruirity for CPU in this generation, mainly batch processing operating system was used. Punch card, paper tape, and magnetic tape was used as input and ouput device.


2nd Generation

Transistors were used in this generation that were cheaper, consumed less power, more reliable and faster. In this generation magnetic cores were used as primary memory. In this generation assembly language and high level programming languages like Fortran, Cobol


3rd Generation

The computers of third generation used integrated circuits in place of transistors. In this generation remote processing, time sharing, multiprogramming operating system were used.


4th Generation

Computer of fourth generation used Very Large Scale Integrated (VLSI) circuits. VLSI have about 5000 transistors with their associated circuits on a single chip made it possible to have micro computers. As a result it gave rise to Personal Computer (PC). All the high level languages like c, c++, DBASE etc., were used in this generation.


5th Generation

This generation is based on parallel hardware and AI(Artificila Intelligence) software.All the high-level language like C++,Java,Net,etc,are used in these generation.
